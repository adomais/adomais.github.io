Typora Markdown 语法说明

[toc]

前言

本文为作者在学习Typora的使用时，为加深记忆以及起到一个记录作用，现学现卖而成的。

全文参考RenS_的Typora Markdown手册。

标题

注意：#后面有一个空格

引用

先输入’>’后，再输入内容会生成一个块引用

代码条/段

输入``` `，在中间输入内容即可，像这样

hello world!

代码块

输入```` 后按下enter`。

表格

输入|第一列|第二列|+ enter创建一个表格

第一个列标题 第二个列标题

列表

  1. 使用*+空格+文本，创建一个无序的列表，*可以用+或-代替
  2. 使用1.+ 空格+文本，创建一个有序的列表，效果如此处所示

复选框

  • 使用 -+空格+ [+空格+ ]，创建一个复选框，效果如此处所示

分割线

在空白行中输入***或--- + enter生成分割线

目录

输入[toc]+enter即可创建目录，提取标题并自动更新

效果如上所示

URL地址

使用<地址>，插入URL地址

例如我的博客:https://adomais.gitee.io/
